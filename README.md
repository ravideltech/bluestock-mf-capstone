# Mutual Fund Analytics

## Capstone Project 1 - Day 1 Data Ingestion and ETL

### Objective

To set up the project environment, load mutual fund datasets, fetch NAV data from APIs, and perform initial data exploration.

### Tasks Completed

* Created project folder structure
* Installed Python dependencies
* Created requirements.txt
* Loaded and analyzed 10 CSV datasets using Pandas
* Printed shape, dtypes, and head() for each dataset
* Fetched live NAV data from MFAPI
* Saved API data as CSV files
* Fetched NAV history for 5 additional mutual fund schemes
* Created data quality summary
* Uploaded project to GitHub

### Technologies Used

* Python
* Pandas
* NumPy
* Requests
* Jupyter Notebook
